# Комманды git


## git status
### состояние репозитория

## git add file.py
### добавить в индекс file.py

## git add .
### добавить все файлы в этом каталоге

## git commit -m "comment for commit"
### создать коммит с комментарием

## git push
### залить изменения с локального репозитория на github

## git pull
### скачать измениния с github

## git remote
### управление связью между текущим и удаленным репозиторием

## git log --oneline
### упращенная история комитов
